git的一个小坑

最近从学校回来了 想在github拉取一些项目做 但是一直报ssh: connect to host github.com port 22: Connection timed out这个错误 具体原因是各个运营商对端口22进行屏蔽了 电信好像好很多 在网上收到了一些做法

运行这段命令，看看是否有成功提示，如果成功，则可以使用这个解决方案
[code]
ssh -T -p 443 git@ssh.github.com
[/code]

简单地配置一下，让你每次ssh连接github都通过443端口。如果你使用Linux，在~/.ssh/config内，添加这些内容，指明ssh连接git@github.com或git@ssh.github.com走443端口。

然后在git bash 中输入 vim ~/.ssh/config

进入编辑模式

然后将下面的代码复制进去
[code]
Host github.comHostname ssh.github.comPort 443User git
[/code]

然后用vim命令，输入：进入编辑模式 输入wq 保存修改即可完成
